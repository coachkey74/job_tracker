# Job Application Tracker

A simple frontend job tracking application built with HTML, CSS, and JavaScript.  
This project allows users to add, display, and delete job applications dynamically.

## Features

- Add job applications with company name and status
- Dynamic UI updates using JavaScript
- Delete functionality using event delegation
- State management with arrays and objects
- Responsive layout using Flexbox

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- JavaScript (ES6)
- Git & GitHub

## What I Practiced

- DOM manipulation
- Event handling with addEventListener
- Preventing default form behavior
- Managing application state
- Dynamic rendering of UI elements
- Filtering arrays for delete functionality

## Future Improvements

- Add localStorage persistence
